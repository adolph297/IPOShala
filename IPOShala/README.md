# IPOShala Frontend

The frontend for the IPOShala project, providing a modern interface for users to browse and monitor IPOs.

## Tech Stack

- **Framework**: Vite + React
- **Styling**: Tailwind CSS
- **State Management**: React Hooks
- **Icons**: Lucide React / Heroicons (if used)

## Key Pages

- **Home**: Overview of active and upcoming IPOs.
- **IPO Details**: Comprehensive view of a single IPO, including bidding centers, documents, and company description.
- **Closed IPOs**: Archive of past IPO listings.

## Development

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Configuration

Environment variables can be set in `.env` (check `.env.example` if available).
